Пт, 10/04/2020 - 22:28 — VTlyusten

1. Поиск простых. Написать полную программу нахождения первых N простых чисел методом перебора, на основе использования отдельной функции, которая распознаёт простые числа.

2. Быстрая сортировка (ваш долг). Чтобы отсортировать массив длины n достаточно отсортировать первые n-1 его элементов, после чего вставить n-й элемент в отсортированную часть так, чтобы упорядочение не нарушилось.
Замечание: для вставки использовать отдельную подпрограмму.

Даю решение на Паскале.
Описать процедуру сортировки, которая реализует этот алгоритм, т.е. изложить на Паскале сказанное на русском языке совсем просто:

  1. procedure qckSrt(var x:ArT; L:Byte);
  2. Begin
  3.       if L>1 then Begin
  4.         qckSrt(x,L-1);//сортируем первые L-1
  5.         elemIn(x,L)//вставляем x[L]на место  
  6.       End
  7. End;

Код процедуры, вставляющей последний элемент массива в уже упорядоченную последовательность тоже прост (кстати, этот алгоритм можно ускорить. Как?):
  1. procedure elemIn(var x:ArT; k:Byte);
  2.    {вставка x[k] в k-1 упорядоченных элементов x}
  3.       var i:Byte;
  4. begin
  5.       for i:=k downto 2 do begin
  6.         //в результате перестановок найдёт своё место:
  7.         if x[i-1]>x[i] then swap(x[i-1],x[i]);
  8.       end;
  9. end;

Внимание! Можно публиковать здесь. Но решения должны быть протестированы!
